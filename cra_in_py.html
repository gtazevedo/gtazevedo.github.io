I won't go into much detail here nor replicate the codes as they were written in .ipynb file format in [this project](https://github.com/gtazevedo/cra_mtae_sas_in_py), feel free to access it and check the notebooks. I'm still reading the book and writing the codes, as I advance I'll keep updating the github project and this page.
